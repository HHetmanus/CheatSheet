Forms Cheat-Sheet

This sheet is to help me to find informations on different types of Input Forms.

I would like to present it in the form of the Table and a Gallery.

- Forms-Input Types -
Tags Function Description
“ Text ” Input Text The normal text inputting field
“ password ” Type password This is encrypted and will not show text
“ number ” Type the number Allows you to type numbers 0-9
“ range ” Type the number Creates the slider within specified range
“ checkbox ” tick in/out allows to tick in/out multiple boxes
“ radio ” tick in/out allows to tick in/out only 1 box
“ submit ” Submit button Creates a submit button
< select > Dropdown List Choose from one of the pre-defined options
< datalist > Datalist Choose or type new option
